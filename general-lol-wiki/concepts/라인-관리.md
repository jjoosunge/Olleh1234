# 라인 관리 (Lane Control)

> 미니언 웨이브를 조절해 교전 위치와 주도권을 통제하는 기술. 라인전 실력의 핵심.

## 🔒 고정 (메커니즘 — 패치로 거의 안 변함)

### 핵심 원리
- 초반 교전 위치는 양 진영 미니언이 만나는 지점에 형성됨 → **미니언 수를 조절해 교전 위치를 옮기는 것**이 라인 관리.
- **원거리 미니언은 근접보다 체력이 낮고 공격력이 ~2배** → 푸시하려면 적 원거리 미니언을 먼저 잡는다.
- 미니언은 일정 속도로 계속 나오므로, 합류가 늦는 쪽이 유리하게 싸우도록 **원거리 미니언 2마리 우위**를 유지하면 라인이 굳는다(프리징 원리).
- **정석: 시야가 확보되면 밀고, 적 정보가 없으면 당긴다.** (단 시대·포지션별 요구는 변동 → [[포지션]])

### 3대 기본 + 응용
| 방식 | 핵심 |
|---|---|
| **밀기(Push)** | 적 원거리 미니언 우선 처치로 전진. 주도권·레벨·골드↑, 적 포탑 압박. 단점: [[갱킹]]·[[로밍]] 취약(천천히 밀어 빅웨이브로 갱 억제) |
| **당기기(Pull)** | 막타만 먹어 적 미니언이 많아지게. 견제 맞아주며 아군 미니언 어그로 유도. 안전(아군 타워 근처)하나 성장 더딤 |
| **프리징(Freezing)** | 원거리 미니언 2마리 우위로 라인 고정. 안전 파밍·[[파밍|디나이]] |
| 느리게 밀기(Slow Push) | 근접 3마리만 빠르게 정리·원거리는 막타 → 빅웨이브 누적. 상대 라인 묶기, 다이브·갱 리스크↓ |
| 홀딩(Holding) | 당기기+프리징. 주도권 있을 때 상대에게 'CS냐 안전이냐' 이지선다 강요 |
| 클리어(Clear) | 라인 위치 무관 빠른 정리(당길 때도 가능). 중반 이후 주류, '초기화'용 |
| 태우기 | 적 부재 시 아군 미니언을 적 포탑에 밀어넣어 골드·경험치 소멸 + 포탑 깎기([[스플릿-푸시]] 선호) |
| 초기화 | 최초 교차 지점으로 라인 복원 |
| 디나이(Deny) | 상대 파밍 차단. 롤은 직접 디나이 불가 → 느리게 밀기·프리징으로 간접 |
| 다이브 | 라인을 적 포탑에 밀어넣고 공성 챔프 처치. [[정글링|정글]] 호응 시 성공률↑ |
| 죽밀라 | 죽더라도 밀어야 유리한 상황(forced wave crush) |

(이상 출처: [[2026-05-16-전략-요약]])

## 유지력 (Sustain)

라인에서 체력·마나 자원을 안정적으로 유지하는 능력. 회복 수단이 있거나 마나를 안 쓰는 챔피언이 유지력이 좋다고 평가되며 라인전 능력과 직결된다. (출처: [[2026-05-16-롤-용어집-요약]])

## 타워 활용

- **타워 허깅**: 타워를 끼고 방어적으로 버티며 후반 도모. 타워 밑 CS 수급 기술 필요(전사 미니언=타워 2방+평타, 마법사=평타+타워 1방+평타).
- **타워 다이브**: 타워 허깅하는 적에게 뛰어들어 잡기. 실드·회복·무적기·어그로 분산 또는 정글/동료 호응 필요. (출처: [[2026-05-16-롤-용어집-요약]])

## 🔄 변동 (패치 의존)
- 🔄 시대·포지션별로 요구되는 라인 관리 방식이 달라짐 — 평가는 패치 시점과 함께. (출처: [[2026-05-16-전략-요약]])

## 연결

- 빠른 클리어 → [[로밍]]/오브젝트 권한. [[아리]]의 승리 조건이 이 연쇄에 의존.
- 파밍 세부는 [[파밍]], 포지션별 적용은 [[포지션]] 참조.
