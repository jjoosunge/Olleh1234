# 🗡️ 아이템 시스템 (Item System)

> 골드로 사는 영구 성장 수단. 적합한 아이템을 얼마나 빨리 완성하느냐가 핵심 플레이. 효과의 종류·체계는 🔒, 가격·메타는 🔄.

## 🔒 고정 (메커니즘 — 패치로 거의 안 변함)

### 구매·슬롯 규칙
- 아군 본진 상점(기본 단축키 **P**)에서 구매·판매. **되팔기 = 구입가 70%** (소모품·시작템은 40%). 되돌리기는 게임에 영향을 줬거나 제단을 벗어나면 불가; 상점 나가기 직전 거래는 자동 취소.
- **아이템 슬롯 7개, 4번은 장신구 전용 → 실질 6개.** 오른 패시브는 1번 슬롯 전설템을 강화.
- 레벨·챔프 스택·오브젝트 버프를 제외하면 **영구 성장의 유일 수단** — 챔피언 강함에 막대한 영향.
- 고유(unique) 제한: 시작 아이템·방어구 관통력·생명선 등 5종 계열은 한 종류만 보유 가능.

### 분류 체계
| 분류 | 정의 |
|---|---|
| 소모 - 장신구 | 4번 슬롯(시즌4 도입). 협곡=시야템(투명/망원/예언자 렌즈·전령의 눈) |
| 소모 - 물약/영약 | 체력 물약(5개), 충전형 / 영약은 9레벨+ 1종만 |
| 소모 - 제어 와드 | → [[와드]] (시야) |
| 신발·장화 | 1티어 속도 / 2티어 부가효과. **1개만 보유**. 사실상 필수 |
| 시작 아이템 | 500골드 내 구비, 판매 40%. 도란 검/반지/방패·세계 지도집·정글 시작템·암흑의 인장(성장형) |
| 기본 아이템 | 조합 재료, 다중 구매 가능. **가성비 기준**(롱소드 350G=AD10 → AD1=35G) |
| 서사급 | 기본 조합 중간 단계 |
| 전설급(코어템) | 상위 조합이 없는 완성 아이템 |
| 특수 / 삭제 | 별도 범위 (미인제스트) |

### 문화
올드 유저 영문 약칭: 트포(삼위일체)·블클(칠흑의 양날도끼)·라위(최후의 속삭임)·인피(무한의 대검)·가엔(수호천사) 등. 근래는 번역명 약칭(정령·태불방·주포) 대세.

## 🔄 변동 (패치 의존 — 소스 시점 2026 기준)

- 🔄 **2026 퀘스트 시스템**: 원거리 딜러는 신발 포함 조건으로 아이템 **7개**, 서포터는 제어 와드 포함 7개 보유 가능.
- 🔄 모든 아이템 가격·스탯·고유 효과 — 패치마다 변동(정밀값은 per-아이템 소스 필요).
- 🔄 아이템 체계 역사: 2021 신화 아이템 도입 → 2024 시즌 신화 폐지 등 대격변(시즌별). 메타 아이템과 궁합 좋은 챔피언은 티어 상승, 템트리 경직 시 약점.
- 🔄 장신구 교체 쿨(5.1~8.8~ 패치마다 변동) — 현재 절반 쿨.

## 🔒 카테고리별 상세 구조 (세부 7종 소스 통합)

> 개별 아이템의 수치·가격·효과는 🔄(패치마다 변동, per-아이템 소스 필요) — 여기서는 변하지 않는 **카테고리 구조**만.

- **시작 아이템**: 도란 계열(검=AD·반지=AP/마나·방패=체력/유지) + 소모품(체력/충전형 물약) + 정글 시작템 + 서폿템 + 암흑의 인장(성장형).
- **소모품**: 물약(체력·충전형) / 장신구(투명 와드·예언자의 렌즈·망원형 개조) / 제어 와드 / 영약(9레벨+).
- **장화**: 1단계(기본 장화·약간 신비한 신발) → 2단계(광전사 군화·마법사 신발·명석함의 아이오니아·판금·헤르메스·기동력·삼중 장화 등) 부가효과 분기. 1개 제한.
- **기본 아이템**: 곡괭이·단검·롱소드·루비 수정·마법무효화의 망토 등 — 조합 재료·가성비 기준.
- **서사급**: 처형인의 대검·흡혈의 낫·광휘의 검·탐식의 망치 등 중간 조합 단계.
- **전설급(코어템, 2024 체계)**: 상점 **역할군 분류**(전사/마법사/암살자/탱커/서포터/원거리)로 정렬. 전사=체력+AD+스킬가속, 등. "1코어=전설 1개". 같은 전설 중복 불가, 일부는 하위 서사 충돌. → 빌드 = 어떤 전설을 어떤 순서로.
- **특수**: 챔피언 전용(갱플랭크 화약통·오른 간이대장간·칼리스타·파이크·피들스틱) · 영감 룬 연계 아이템 · 특정 맵 한정.
