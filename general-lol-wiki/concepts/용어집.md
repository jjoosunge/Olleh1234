# 📖 용어집 (Glossary) — LoL 용어 색인 허브

> LoL 용어를 영역별로 정리한 **레퍼런스 허브**. 깊이가 필요한 항목은 전용 개념 페이지로 분리하고 여기서 링크한다.
> 출처: 전 항목 (출처: [[2026-05-16-롤-용어집-요약]])

## 1. 공용·일반

| 용어 | 뜻 |
|---|---|
| 캐리(Carry) | 게임을 승리로 이끄는 역할/행위. 후반 포텐셜 강한 챔피언=캐리형. 막기 쉬운 챔프=안티캐리. 후반 힘 빠지는 챔프=유통기한(fall-off) |
| AD/AP | 공격력(Attack Damage)/주문력(Ability Power). 주 계수에 따라 AD·AP·하이브리드 챔피언 |
| AR/MR | 방어력(물리 경감)/마법 저항력(마법 경감). 고정 피해(트루뎀)는 경감 불가 |
| CS(Creep Score) | 미니언·정글몹 막타 처치 수. 분당 10 이상이 매우 높음. → [[파밍]] |
| 스노우볼링 | 초반 이득을 굴려 격차를 키워 후반을 압도 |
| 피딩·헌납 | 상대에게 킬·버프를 넘김. 한국선 '키우다'. 비꼬아 헌납·배달 |
| 갱킹 vs 로밍 | **정글러 라인 개입=[[갱킹]]**, **라이너/서포터 타 라인 개입=[[로밍]]** |

## 2. 역할군 → [[역할군]]

라이엇 공식 6종(암살자/전사/원거리/서포터/마법사/탱커) vs 유저 통용(탱/딜탱/메이지/암살자/원딜/서포터). 공격방식 분류: 원거리·근거리·캐스터(메이지)·누커(암살자)·딜탱(브루저).

## 3. 맵 오브젝트 → [[오브젝트]]

포탑/타워 · 미니언 · 넥서스 · 억제기(슈퍼 미니언 억제) · 우물(넥서스 오벨리스크, 초당 2000 순수 피해) · 정글 · 바론(내셔 남작) · 드래곤 6종+장로 · 버프몹(레드/블루).

## 4. 라인 운영 → [[라인-관리]]

유지력(Sustain) · [[파밍]](더티/오버 파밍) · 딜교환(Trade) · 푸시/클리어/프리징 · 디나이(Deny) · 타워 허깅 · 타워 다이브 · 죽밀라(forced wave crush).

## 5. 정글 운영 → [[정글링]]

정글링 · 리시(Leash) · 카운터 정글링([[갱킹|카정]]) · 커버 플레이 · 강타 싸움(Smite Battle).

## 6. 팀 전략 → 개별 페이지

| 용어 | 뜻 |
|---|---|
| [[인베이드]] | 미니언/몹 생성 전 상대 진영 난입 교전 (30초 미니언·55초 몹 기준) |
| [[스플릿-푸시]] | 인원 분배로 다수 라인 동시 압박 (1-3-1). 백도어와 구분 |
| [[이니시에이팅]] | 교전 개시(engage). 메이킹·존야시에이팅 포함 |
| [[카이팅]] | 간격 벌리며 평타 — → [[카이팅]] (스텁) |
| 닷지(Dodge) | 챔피언 선택창 이탈로 방 폭파. LP 페널티(3→10) |
| [[조닝]] | 구역을 위협해 상대 접근 차단 (거리 조절) |
| [[페이스체크]] | 시야 수단 없이 부쉬/사각 직접 진입 — 고위험 |
| 던지기(Throw) | 판단착오/고의로 게임을 내주는 플레이 |
| 스펠체크 | 적 점멸·궁·오브젝트 쿨을 채팅으로 기록 |
