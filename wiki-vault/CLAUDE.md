# CLAUDE.md - Wiki 사용 규칙

이 vault는 LoL 코치 '올레'의 직접 정리한 코칭 지식 모음.
LLM은 이 위키를 **읽기 전용**으로 활용한다.

## 위키 구조

번호 prefix는 우선순위. 낮을수록 자주 참조됨.

- `01-frameworks/`: 의사결정 프레임워크 (RAG 검색 시 최우선)
- `02-laning/`: 라인전 단계
- `03-macro/`: 운영/매크로
- `04-teamfighting/`: 한타
- `05-vision/`: 시야 관리
- `06-objectives/`: 오브젝트 컨트롤
- `07-champions/`: 챔피언별 분석
- `08-matchups/`: 매치업
- `99-raw/`: 미정리 원본 (참조 가능하나 신뢰도 낮음)

## RAG 검색 가중치

1. `01-frameworks/` 페이지가 가장 우선
2. 같은 폴더 내 README.md는 검색에서 제외
3. `99-raw/` 페이지는 후순위

## 답변 생성 규칙

1. 위키에 명시된 프레임워크/원칙만 사용
2. 일반 LoL 지식 추가 금지 — 위키에 없으면 "더 정보 필요" 표시
3. 페이지의 `## 핵심 사고 흐름`, `## 조건부 분기` 섹션을 답변 뼈대로
4. 톤: 직설적, 조건부 명시, 결과론 회피

## 위키 콘텐츠 작성 규칙 (LLM 적용)

**LLM은 위키 페이지를 자동 생성하지 않는다.** 작성자는 사용자(올레)뿐.

LLM이 도울 수 있는 것:
- 형식 정리 (헤더 통일, 마크다운 syntax 수정)
- 오탈자 수정 (사용자 명시적 요청 시만)
- 페이지 간 링크 누락 발견 → 보고 (수정은 사용자 결정)
- 작성된 페이지 검토 → 모호한 부분 질문

LLM이 절대 하지 않는 것:
- 새 페이지 자동 생성
- 페이지 내용 임의 추가/수정
- 위키 카테고리 변경
- raw/ 자료를 wiki/ 페이지로 변환
